# TranscriptG - Free YouTube Transcript Generator & AI Summarizer

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)
[![TypeScript](https://img.shields.io/badge/TypeScript-5.0-blue.svg)](https://www.typescriptlang.org/)
[![React](https://img.shields.io/badge/React-18-blue.svg)](https://react.dev/)

**TranscriptG** is a high-speed, server-backed web application designed to convert any public YouTube video, Shorts, or live stream into clean, fully punctuated text transcripts. Beyond basic subtitle grabbing, TranscriptG features an integrated suite of content utilities: an AI-powered YouTube Video Summarizer, a timestamped video chapters generator, a YouTube-to-blog post markdown editor, a viral short script clipper, and a sandbox for SEO keyword extraction.

🚀 **Live Website & Tool:** [https://transcriptg.com/](https://transcriptg.com/)

---

## 🎯 Key Features

- **Instant Transcript Grabbing**: Convert any public YouTube URL into clean, readable text instantly.
- **AI-Powered Summarization**: Generate high-quality summaries, key takeaways, and structured chapters.
- **Content Repurposing Suite**: Turn video transcripts into formatted blog posts, social media scripts, and newsletters.
- **Multi-Language Support**: Translate transcriptions into over 15+ major global languages.
- **Completely Free**: No registration, no limits, and no hidden subscriptions.

## 🛠️ Tech Stack & Architecture

- **Frontend**: React (Vite), TypeScript, Tailwind CSS, Lucide Icons, Framer Motion
- **Backend**: Node.js, Express, YouTube Transcription API proxies, and Gemini AI integration
- **Linting & Quality**: ESLint, TypeScript compilation check

## 🚀 Local Development

Follow these steps to run the project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/akashsinghsolanki66/transcriptg.git
   cd transcriptg
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env` file in the root directory and add any required API keys (such as `GEMINI_API_KEY` for AI features).
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```
   The application will be accessible at `http://localhost:3000`.
